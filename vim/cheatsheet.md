# Vim Key Bindings

## Folds

|Key|Usage|
|-|-|
|`za`|Toggle single fold|
|`zA`|Toggle fold recursively|
|`zo`|Open single fold|
|`zO`|Open fold recursively|
|`zc`|Close single fold|
|`zC`|Close fold recursively|
|`zM`|Close all folds|
|`zR`|Open all folds|


## NERDTree

|Key|Usage|
|-|-|
|`m`|Modify filesystem (create files, directories, ....)|
